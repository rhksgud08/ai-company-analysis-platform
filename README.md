# AI 기반 기업 재무·ESG 비교 분석 플랫폼

## 프로젝트 소개

이 프로젝트는 기업의 재무정보와 ESG 정보를 한 화면에서 비교하고,
기업 분석에 AI를 활용할 수 있는 가능성과 한계를 탐구하기 위해 제작한다.

## 탐구 배경

기업의 사업보고서와 지속가능경영보고서는 분량이 많고
재무정보와 ESG 정보를 함께 비교하기 어렵다.

이에 따라 핵심 재무지표와 ESG 지표를 정리하여
두 기업을 비교할 수 있는 웹 기반 플랫폼을 설계하고자 한다.

## 주요 기능

- 두 기업 선택 및 비교
- 매출액, 영업이익, 당기순이익 비교
- 재무비율 자동 계산
- 탄소배출량 및 ESG 공시 내용 비교
- 기업별 강점과 위험요인 정리
- AI 분석 결과의 활용성과 한계 평가

## 활용 자료

- 금융감독원 전자공시시스템 사업보고서
- 기업 지속가능경영보고서
- 기업 공식 홈페이지와 공시자료

## 개발 환경

- Python
- Streamlit
- Pandas
- GitHub

## 주의사항

본 프로젝트는 교육 및 탐구 목적으로 제작하며,
실제 투자 판단이나 투자 권유를 목적으로 하지 않는다.
